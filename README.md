A simple Finance analysis tool I made to analyze how many expenses and income happen at some period of time. With additional statistics information, there is also --help command if you are confused that explains whats the statistical information is about.

# Install
-> `git clone https://github.com/3oFiz4/simple_finance_presenter`
-> `python main.py transc.txt`

# Template in transc.txt
The file name may be vary. But the acceptable template are this:
```md
balance=500 // initial balance
limit=400 // budget limit, red flag is beyond this.

# date
price - label/name - desc
...


# 4/1/26
10 - Ice Cream - so tasty!
4 - Cola - refreshing
+20 - Found money - lucky day

# 4/2/26
25 - Groceries - weekly
8 - Coffee - morning fix
+100 - Freelance - web project

# 4/3/26
50 - Gas - full tank
15 - Parking - downtown
100 - Car repair - brake fix

# 4/5/26
35 - Movie - date night
2 - Candy - snack
+50 - Cashback - CC reward
```
